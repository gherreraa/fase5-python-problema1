## Problema seleccionado

Problema 1: Evaluación del nivel de compromiso de sesiones de clientes.

## Descripción

Este programa en Python utiliza una matriz con datos de sesiones de clientes, compuesta por ID del cliente, duración en segundos y cantidad de clics. A partir de estos datos, clasifica cada sesión en nivel de compromiso Alto, Medio o Bajo.
